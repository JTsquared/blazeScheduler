# Privacy Policy for Blaze Stream Custom Filters

**Last Updated:** February 26, 2026

## Overview

Blaze Stream Custom Filters is a browser extension that helps you organize and filter streamers on Blaze.stream using custom tags.

## Data Collection

**We do not collect, transmit, or share any personal data.**

## Data Storage

All data created by this extension is stored locally in your browser using Chrome's storage API:

- **Custom filter names** - Tag categories you create (e.g., "Favorites", "Gaming")
- **Streamer tags** - Your associations between streamers and filters

This data:
- Never leaves your device
- Is not sent to any external servers
- Is not accessible to the extension developer
- Remains private to your browser profile
- May sync across your devices if you have Chrome Sync enabled (handled by Google)

## Permissions Used

The extension requires the following permissions:

- **storage** - To save your custom filters and streamer tags locally in your browser
- **host_permissions (https://blaze.stream/*)** - To add filtering features and tag buttons to the Blaze.stream website

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

## Data Deletion

You can delete all extension data at any time by:
1. Uninstalling the extension, OR
2. Clearing your browser's extension data in Chrome settings
